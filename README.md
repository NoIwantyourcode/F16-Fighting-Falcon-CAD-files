# What exactly is this project?

This project is an RC plane based on the F16 fighting falcon that I am designing and making, It is designed to be made completely out of foam and hot glue!

# Why I made this project?

I made this project as I have a passion in Aeromodelling and engineering and have a dream of being an aeronautical engineer in the future! What better way to represent that currently than to make an RC plane!

# Does this project have any firmware?

This project does not need any firmware as it uses ready made components which I already own such as the transmitter and reciever
